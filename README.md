Starbucks Smart Choice MX
2. Objetivo del sitio web
El objetivo del sitio web es desarrollar una aplicación web interactiva que ayude a los
usuarios a elegir bebidas y alimentos de manera rápida y sencilla.
El sistema mostrará diferentes preguntas relacionadas con los gustos del usuario, como si
prefiere bebidas frías o calientes, con café o sin café, o si desea ciertos ingredientes.
Con base en las respuestas del usuario, el sistema generará una recomendación
personalizada de un producto disponible en Starbucks, mostrando una imagen y una breve
descripción.
La finalidad es mejorar la experiencia del usuario al momento de elegir un producto,
facilitando la toma de decisiones y haciendo el proceso más rápido e interactivo.
